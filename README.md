HAT for Raspberry PI 4 with isolated rs232 , rs485 , CAN-FD

This board was made for personal use with other projects.
outputs for manufacturing:
1) gerbers and drill files
2) BOM for jlcpcb store
3) pick and place file

Board capabilities
1) isolated RS232 on standart UART pins (8 , 10)
2) isolated RS485 on additional UART with enable pin (27,28,29)
3) isolated CAN-FD on SPI0 (standard pins with interrupt pin on pin 15)

Production cost on jlcpcb for 5 prototypes is app. 150$
